# GridRecall Features & Highlights

## Core Features

### 1. Customizable Grid Sizes
- Multiple grid configurations from 3x3 to 5x6
- 12 different size options to match player skill level
- Ability to change grid size at any time

### 2. Sequence Creation & Playback
- Create custom sequences by tapping grid buttons
- Step-by-step playback with pause/resume functionality
- Visual highlighting of current step during playback
- Manual navigation through sequence steps

### 3. Modern UI Design
- Sleek, glass-effect interface with subtle animations
- Dark mode optimized for reduced eye strain
- Responsive design that works on all screen sizes
- Subtle visual feedback for all interactions

### 4. Accessibility
- High contrast visual indicators
- Clear, readable text and icons
- Keyboard navigation support
- Screen reader compatible elements
